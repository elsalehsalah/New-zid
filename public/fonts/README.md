# Fonts Directory

Place your custom font files here:

## Codec Pro (Headers)
- `CodecPro-Regular.woff2` (weight: 400)
- `CodecPro-Medium.woff2` (weight: 500)
- `CodecPro-Bold.woff2` (weight: 700)
- `CodecPro-ExtraBold.woff2` (weight: 800)
- `CodecPro-Black.woff2` (weight: 900)

## Suisie (Body)
- `Suisie-Regular.woff2` (weight: 400)
- `Suisie-Medium.woff2` (weight: 500)
- `Suisie-Bold.woff2` (weight: 700)

## Supported Formats
- `.woff2` (preferred)
- `.woff`
- `.ttf`
- `.otf`

## Note
If your font files have different names or formats, update the paths in `app/layout.tsx` accordingly.
